ISI Drone

ISI-Drone was created as cool place to buy drones!that focus to provide exceptional service, fast shipping, and premier products.

This Site is uniqe plateform for those who have a passion for drones. This application not only strive to provide a memorable shopping experience but also enjoy offering knowledge and experience to customers through custom made enterprise solutions.

Admin can sell different products by making then available on Web-application.

Customer can order the products from the Mobile Application.

Application also provides the insight of system by various graphs such as Incoming Purchases, Recent Sales, Most Sold and Least Sold products etc.

Usage:

Eclipse 2019-12 , jdk-8u251-windows-x64, Tomcat 8.0

Installation:

  • clone project using git https://github.com/Gagan1717/ISI-Drone.git

Technologies used:

  • Jsp
  • Servlet
  • My Sql Server
  • Jenkins.
