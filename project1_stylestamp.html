Style Stamp-Final Stamp to your Fashion

Style Stamp is an E-Commerce Application which provides facility to buy and sell fashion accesories.

Admin can sell different products by making then available on Web-application.

Customer can order the products from the Mobile Application.

Application also provides the insight of system by various graphs such as Incoming Purchases, Recent Sales, Most Sold and Least Sold products etc.

Style stamp the style stap project is e commerce application that allows shop keeper to manage categories of product and products via admin portal
