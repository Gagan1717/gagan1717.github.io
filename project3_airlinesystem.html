Airline Reservation system

Airline Reservation System is an E-Commerce Application that facilitiate its users to manage their airline ticket reservation.

Customer can Book , Cancle and Update his airticket.

Application also gives information about the market competition factors such as price and services.

With all this airlines safety measures are also described in this application.

Installation:

  • clone project using git https://github.com/Gagan1717/Airline-Reservation.git

Technologies used:

  • Asp.Net
  • PHP
  • MySQL
